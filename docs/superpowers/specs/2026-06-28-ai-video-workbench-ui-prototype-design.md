# AI 视频工作台 UI 原型设计

## 目标

把当前工程样板式页面升级成可演示的短视频生产工作台原型：一眼能看出这是 AI 短视频 SaaS，而不是普通后台表单。

## 参考但不照抄

参考超级编导的信息架构：功能入口、AI 成片、素材库、批量剪辑、爆款复刻、任务队列。视觉上不照抄，采用“AI 运营驾驶舱 + 剪辑工作台”的科技风。

## 第一版原型范围

只改前端展示层，不改后端接口和数据结构。

- 首页：展示核心能力入口、积分/任务/模型状态、最近任务。
- 工作台：左侧模块导航，中间手机竖屏预览和脚本候选，右侧 AI 参数面板，底部批量任务/时间线感觉。
- 保留现有登录、客户资料、模板、AI 文案生成能力。
- 用静态视觉增强已有数据，不新增真实剪辑逻辑。

## 风格

- 深色背景
- 蓝紫霓虹渐变
- 玻璃拟态卡片
- 竖屏手机预览
- 数据流/扫描线感
- 清晰的“生成脚本 -> 确认 -> 制作视频”流程

## 验收

- 用户打开客户端第一眼能理解产品定位。
- 文案生成入口比旧版更明显。
- 页面仍能跑现有前端测试和 build。
- 不引入新依赖。
